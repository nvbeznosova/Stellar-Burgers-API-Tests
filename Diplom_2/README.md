# Diplom_2

## Описание
Проект предназначен для автоматизированного тестирования API.  
Реализованы тесты для проверки корректности работы эндпоинтов, а также вспомогательные утилиты для запуска и анализа результатов.

## Структура проекта

Diplom_2/
│
├── tests/                # Папка с тестами
│   ├── test_user.py      # Тесты на создание пользователя
│   ├── test_order.py     # Тесты на создание заказа
│   └── test_login.py     # Тесты авторизации
│
├── conftest.py           # Общие фикстуры и настройки для pytest
│
├── requirements.txt      # Список зависимостей проекта (pytest, requests и др.)
│
├── .gitignore            # Файлы и папки, исключённые из Git (venv, __pycache__)
│
├── helpers.py            # Вспомогательные функции (например, генерация случайного email)
│
├── urls.py               # Содержит эндпоинты API
│
└── README.md             # Описание проекта

## Найденные баги / Несоответствия API

- **Создание заказа без авторизации**
  - Ожидаемый результат: при попытке создать заказ без заголовка `Authorization` должен возвращаться статус `401 Unauthorized`
  - Фактический результа : API возвращает `200 OK` и заказ создается как гостевой
  - Статус: баг, требует уточнения у разработчиков
